Windows desktop app

Organise files locally, review every move, and keep rollback one click away.

urordo is built for people who want a calmer file system without handing their folders to a remote service. It scans locally, groups by meaning, lets users approve the plan, and keeps a full activity trail.

How it works

01

Pick a folder

Choose a messy folder or set one up as a watcher. urordo scans locally and classifies files by meaning.

02

Review the plan

Check every move, adjust categories, and keep ignore rules in one place before you apply anything.

03

Apply and roll back if needed

Moves are logged, history stays visible, and rollback is available from the Activity screen.

What users get

  • Manual review for trust-sensitive organising
  • Recursive watchers for incoming folders
  • Optional Gemini help through a user-owned API key
  • Installer-based distribution with no localhost requirement

Install flow

  1. Download the installer from this page
  2. Run setup and open urordo
  3. Complete onboarding and choose a source folder
  4. Optionally add Gemini later from Settings
