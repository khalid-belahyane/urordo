## 📖 Overview

**urordo** is an intelligent, local-first file organizer for Windows that doesn't just read file extensions—it understands meaning. 

By analyzing folder structures, document metadata, and file taxonomy, urordo categorizes files into meaningful domains autonomously. Users retain total control via an absolute "Review First" workflow: urordo scans, classifies, and proposes a new structure. Only after explicit approval are the files moved.

<br />

## ⚡ Core Features

- **🛡️ Review First Workflow:** See every single proposed move through an intuitive UI before anything changes.
- **⏪ Time-Machine Rollbacks:** Every operation is cryptographically logged. Made a mistake? Roll it back precisely from the Activity screen.
- **🔐 Absolute Privacy (Local-First):** Your files, paths, and patterns never leave your machine. No accounts. No tracking. Nothing.
- **🤖 Smart Mode (Opt-in):** Add your own Google Gemini API key to leverage AI for ultra-precise classification without relying on vendor lock-in.
- **📡 Background Watchers:** Assign "Drop Zones" for recursive folder watching, securely routing downloads precisely where they belong automatically.
