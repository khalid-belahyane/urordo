# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.0.1   | :white_check_mark: |
| < 0.0.1 | :x:                |

## Reporting a Vulnerability

Security is a high priority for this project. If you discover a vulnerability, please report it privately.
**Do not** file a public issue for security findings.

1. Please email the maintainer or use the private reporting functionality directly if available.
2. Provide precise replication steps and the impact of the bug.

We aim to evaluate all disclosures within 72 hours.
